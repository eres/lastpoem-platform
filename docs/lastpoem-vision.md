# LastPoem Vision

LastPoem is a portable publishing infrastructure for poets.

The first living case is ERESpoet.com: a poet eStand, EPUB 3 reader, publishing workflow, and commerce layer for poetry books.

Principle: poetry does not lack readers; it lacks distribution architecture.

MVP:
- Poet profile
- Book landing page
- EPUB 3 reader
- Reservations / commerce
- Reader comments and ratings
- AI editorial assistant

Architecture principle:
Build open, portable, self-hostable components wherever possible.
