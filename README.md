# LastPoem Platform

LastPoem is an open, portable publishing infrastructure for poets.

This repository begins with ERESpoet.com as the first living case:
a poet eStand, EPUB 3 reader, publishing system, and commerce layer for poetry books.

## Structure

- apps/erespoet — first public implementation
- packages/reader — EPUB 3 reader
- packages/epub-engine — EPUB generation tools
- packages/ui — shared interface components
- docs — architecture, product notes, and vision
